Transmission Fault Line Detection using Unsupervised Machine Learning

Developed a fault detection system for transmission lines using Fourier Transform, Fast Fourier Transform (FFT), LSTM networks, and K-Means clustering. The project analyzes time series voltage signals (VRM, VYM, VBM) by converting them into the frequency domain to extract key fault features. The LSTM model captures temporal dependencies, while K-Means clusters fault signatures into distinct categories. This approach enhances fault diagnosis accuracy and supports predictive maintenance in power transmission systems.
